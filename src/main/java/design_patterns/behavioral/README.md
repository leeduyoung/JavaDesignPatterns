# 행동 관련 패턴

행동 패턴은 개체 간의 통신 또는 개체가 상호 작용하고 책임을 수행하는 방법에 중점을 둔다. 

이러한 패턴은 개체 간의 통신 및 상호 작용을 개선하여 응용 프로그램을 보다 유연하고 확장 가능하며 유지 관리할 수 있도록 한다.
